What's New in LSDK 20.12

Highlights

  • LTS 4.19.139 and LTS 5.4.47 update
  • Support of PREEMPT_RT on LTS kernel 5.4.47-rt28
  • U-Boot v2020.04 update
  • Support of LX2162A QDS
  • Support of PREEMPT_RT on LS1028
  • Dropping the support of LX2160A rev1. Rev2 is supported
  • Removal of Edgescale support
  • ACPI mode on LX2160A and LS1046A
  • MC 10.24.0 update
  • OPTEE 3.8.0 update
  • Refer to Fixed, open and closed issues which has a list of all fixed issues
  • Inclusion of additional workarounds for Chip Errata: A-008098, A-009531, A-010194, A-010305, A-011127, A-011264, A-0011334, A-050426, A-050479

List of changes

Processor and Board Support

  • Support of LX2162A QDS
  • Dropping the support of LX2160A rev1. Rev2 is supported.

NXP LSDK Userland

  • No change since the last LSDK

Linux Kernel Core, Virtualization

  • LTS kernel 4.19.139 update
  • LTS kernel 5.4.47 update
  • Support of PREEMPT_RT on LTS kernel 5.4.47-rt28
  • Support of PREEMPT_RT on LS1028A

Linux Kernel Drivers

  • Support of LX2162A QDS
  • ACPI mode on LX2160A and LS1046A
  • DPAA2: adding bulking to XDP, building Tx S/G frames instead of realloc-ing skb, update bridge link status
  • ENETC: adaptive interrupt coalescing
  • Felix switch: configurable MTU for (jumbo frame support), port and flow policers (DSA)
  • IEEE1588: one-step timestamping on DPAA2, pps and periodical clock signal on Felix switch, FIPER3 on DPAA2, external trigger
  • PCIe: power management support
  • PPFE: driver be built-in instead of module
  • SDHC: eMMC RPBM partition, HS200 on LS1088A

Data Plane Development Kit (DPDK)

  • Support of LX2162A QDS
  • Crypto: SEC PDCP-SDAP format on LX2160A, non-HMAC auth algorithm on LX2160A
  • Memory pool: optimizing the buffer allocation time

Virtualization - OVS-DPDK

  • No change since the last LSDK

Trust Firmware - A (TF-A)

  • TF-A v2.3 on LX2160A and LX2162A

U-Boot Boot Loader

  • U-Boot v2020.04 update
  • Support of LX2162A QDS
  • GPIO on LX2160A
  • SDHC: HS200 and HS400 on LX2162A
  • Watchdog: SBSA on LX2160A

Unified Extensible Firmware Interface (UEFI)

  • ACPI mode on LX2160A and LS1046A

Other Tools and Utilities

  • eIQ machine learning applications update
  • Flexbuild
    • Support of LX2162A QDS
    • No LX2160A rev1 image being released
    • Building LSDK on ARM board with non-root account
    • Supporting secure ARM-COT, updating the composite firmware for LX2160ARDB and LX2162AQDS
    • Configurable boot partition and root partition in flex-installer
    • Integrating mmc-utils for eMMC RPMB partition
    • Supporting secure verified boot
    • PPFE: driver built-in instead of module
  • MC 10.24.0 update. Changes are listed on CHANGELOG
  • OPTEE 3.8.0 update
