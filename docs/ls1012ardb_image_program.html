Image programming instructions of LS1012ARDB

Switch to qspi bank1 and program composite firmware to bank2.

$ wget http://www.nxp.com/lgfiles/sdk/lsdk1709-update-103017/firmware_ls1012ardb_uboot_qspiboot.img
=> tftp a0000000 firmware_ls1012ardb_uboot_qspiboot.img
=> i2c mw 0x24 0x7 0xfc;i2c mw 0x24 0x3 0xf5
=> sf probe 0:0
=> sf erase 0 +$filesize
=> sf write a0000000 0 $filesize
=> reset

Programm individual images(uboot/rcw/ppa/ppa_pfe addresses) on QSPI from bank1.

=> i2c mw 0x24 0x7 0xfc; i2c mw 0x24 0x3 0xf5

RCW

=> tftp 0x80000000 PBL_0x35_0x08_1000_250_1000_default.bin
=> sf probe 0:0
=> sf erase 0 40000
=> sf write 0x80000000 0x0 40000

Uboot

=> tftp 0x80000000 uboot_ls1012ardb_qspi.bin
=> sf probe 0:0
=> sf erase 0x100000 100000
=> sf write 0x80000000 0x100000 100000

PPA

=> tftp 0x80000000 ppa.itb
=> sf probe 0:0
=> sf erase 400000 +$filesize
=> sf write 0x80000000 400000 $filesize

PFE firmware

=> tftp 0x80000000 pfe_fw_sbl.itb
=> sf probe 0:0
=> sf erase a00000 +$filesize
=> sf write 0x80000000 a00000 $filesize
